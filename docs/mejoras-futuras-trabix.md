# Mejoras futuras de contenido y activos para Trabix Granizados

## Objetivo
Documentar mejoras futuras que no hacen parte de la primera reestructuración del sitio, pero que ayudarán a reforzar credibilidad, prueba social y calidad visual conforme exista más material real de la marca.

## Carpeta futura de fotos reales
- Crear una carpeta dedicada para fotos reales de:
  - eventos,
  - activaciones,
  - establecimientos,
  - granizados en contexto,
  - puntos de venta,
  - presencia nocturna.
- Esa carpeta debe pensarse como repositorio visual oficial para reemplazar o complementar piezas promocionales genéricas.
- Las imágenes deben priorizar situaciones reales de uso del producto:
  - personas consumiendo,
  - producto exhibido,
  - barras o mesas de venta,
  - ambiente nocturno o de fiesta,
  - aliados o puntos donde Trabix ya se mueve.

## Uso futuro de esas fotos en la web
- Home:
  - reforzar bloque de credibilidad,
  - mostrar presencia real de marca,
  - conectar la promesa de practicidad con escenas auténticas.
- Mayoristas:
  - reforzar secciones de eventos, reventa y activaciones,
  - mostrar que el producto ya funciona en escenarios reales.
- Emprende con Trabix:
  - apoyar la narrativa de oportunidad con evidencia visual,
  - mostrar movimiento real de marca y presencia comercial.
- Retail:
  - usar imágenes reales para aumentar antojo, cercanía y deseo inmediato.

## Mejoras futuras de contenido
- Añadir más prueba social verificable:
  - nuevos aliados activos,
  - más ciudades cubiertas,
  - tipos de eventos atendidos,
  - nuevos puntos o establecimientos.
- Revisar si conviene incorporar testimonios cortos de aliados o clientes.
- Evaluar una sección futura de preguntas frecuentes por ruta:
  - retail,
  - mayoristas,
  - aliados.
- Evaluar una futura sección de cobertura más precisa cuando el alcance operativo sea más estable.

## Mejoras futuras del funnel
- Definir si más adelante conviene separar aún más el flujo de aliados:
  - emprendedor operativo,
  - persona con capital,
  - punto comercial o establecimiento.
- Revisar si se necesita una automatización más profunda entre la web y el flujo real de registro de aliados.
- Evaluar si conviene incorporar materiales descargables o piezas de apoyo para aliados en fases posteriores.

## Mejoras futuras visuales
- Enriquecer el sistema visual con más composiciones basadas en sabores y fondos reales.
- Integrar galerías o carruseles livianos cuando existan suficientes fotos propias.
- Crear una biblioteca de piezas visuales reutilizables para campañas, promociones y temporadas.
- Ajustar el tono visual por temporada o tipo de evento sin romper la identidad principal.
